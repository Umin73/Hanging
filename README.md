## 게임 소개
  큰 전쟁이 끝난 직후, 세계는 통합되어 정부군과 반란군으로 나뉩니다. <br/>
  정부는 부족한 사람 수를 충족하기 위해 복제인간을 제작했고, 플레이어는 그 결과 탄생한 복제인간입니다.<br/>
  플레이어는 구직을 통해 사형감별사 직업을 얻었고, 플레이하면서 정부군 또는 반란군을 도와 살아남는 게임입니다.


## 기술 스택
  - 게임 엔진 : Unity (2021.3.12f1 LTS)
  - 프로그래밍 언어 : C#
